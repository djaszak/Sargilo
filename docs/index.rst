Sargilo - Test data loader
==========================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   aim
   general/index
   integration/index
   development/index


Sargilo is the esperanto word for "Loader" and that is exactly what it is. This project is a human-friendly test data
loader that reads your test data from a setup file (written in yaml), creates python objects and makes them accessible
in a structured and intuitive way.
